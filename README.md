## 特性

查看 Slimefun 指南获取更多信息。

### 工具与资源
- 灵魂收集
  - 从生物掉落更多经验
  - 有几率掉落浓缩灵魂，一种制作材料
- 光与暗魔法植物
  - 用于酿造药水以及制作部分资源
  - 使用灵魂或魔法精华注入橡树树苗获得

### 合成
- 神圣祭坛
  - 用于制造高级物品
  - 可以制作金属
  - 可以将不同种类的生命变成压缩碳
- 华丽炼药锅
  - 用于酿造强力的药水
    - 仁慈酿造与恶毒混合物
- 注入祭坛
  - 可以向工具、武器、防具注入强力的魔法
  - 每个装备仅能选择一种注入魔法，不能同时添加多种注入魔法
